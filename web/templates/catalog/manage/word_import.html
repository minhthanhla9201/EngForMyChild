{# Hướng dẫn định dạng CSV #}
Định dạng CSV (dòng đầu là tiêu đề):
  • text_en — từ tiếng Anh (bắt buộc)
  • text_vi — nghĩa tiếng Việt
  • topic — tên chủ đề (tiếng Anh); tự tạo nếu chưa có
  • topic_vi — tên chủ đề tiếng Việt (tuỳ chọn)
  • level — độ khó, mặc định 1 (tuỳ chọn)
  • image — đường dẫn hình trong media/, vd images/cat.jpg (tuỳ chọn)
Nhập lại cùng file sẽ không tạo trùng (cập nhật nghĩa/hình nếu đổi). IPA tự sinh khi để trống. Cột image chỉ trỏ tới file — hãy đặt sẵn file hình vào web/media/images/ (nhập không tự tải hình về).
